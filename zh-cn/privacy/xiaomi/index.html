一、适用范围与处理者

本政策专门适用于从小米应用商店分发的紫霞浏览器 Android 应用（应用包名：com.zixia，以下简称“应用”），以及为该版本提供的紫霞在线资源。应用名称为“紫霞浏览器”，开发者及个人信息处理者为海南溪谷科技有限公司。

应用以浏览和本地阅读为核心功能。浏览历史、书签、书架、下载文件等数据主要保存在设备本地；应用不会将这些本地内容用于广告画像，也不会出售个人信息。

敏感个人信息提示：精确位置、相机画面以及你选择交由应用处理的本地文件可能属于敏感个人信息。应用仅在你主动使用对应功能并通过 Android 授权后处理；拒绝授权只会影响对应功能，不影响其他无关功能。

二、同意与最小必要原则

应用首次运行时会通过独立页面提示你阅读本政策和用户协议，并提供明确的“同意并继续”和“不同意”选项。涉及位置、相机、文件等系统权限时，应用还会在具体功能触发时申请 Android 授权。我们按照合法、正当、必要、诚信和最小够用原则处理信息，不会因你拒绝非必要权限而停止提供无关功能。

五、自动联网和基础服务

  • 网页访问：你打开的网站会收到正常联网所需的 IP 地址、请求网址、User-Agent 和该网站获准使用的 Cookie，并按其隐私政策处理。
  • 首页配置与赞助快捷方式：进入默认首页时，应用可能向 browser.zixia.com 请求按语言提供的首页配置及明确标注的赞助快捷方式列表，并向快捷方式目标网站请求图标。请求包含必要网络技术信息，不包含本地历史、书签、书架或下载文件；我们不跟踪赞助链接的展示或点击。
  • 广告过滤规则：首次准备广告过滤或更新订阅时，应用会向规则提供方下载公开规则，默认规则来源可能包括 AdGuard 的 filters.adtidy.org。提供方会收到必要网络技术信息。
  • 搜索建议：该功能默认可能开启。地址栏输入文字时，关键词可能发送至百度、Google 或当前搜索引擎配置的 OpenSearch 建议服务；你可以在设置中关闭。
  • 网站图标：为显示首页或书签图标，应用可能请求相应目标网站的公开图标，目标网站会收到必要网络技术信息。
  • 更新检查：你主动检查或下载更新时，应用可能访问 GitHub API、Google Play 或更新文件服务。

六、可选在线服务

  • 翻译：你选择的文字、图片或网页内容会发送给你选择的翻译服务，例如 Google、DeepL 或 Papago/Naver。
  • AI 助手：你的问题及你要求处理的选中文字或网页内容会发送给你配置的 OpenAI、Anthropic、Google Gemini、Moonshot、智谱 BigModel、DeepSeek 或兼容接口。服务商配置和 API 密钥保存在设备上，并在鉴权时发送给所选服务商。
  • 语音：系统语音使用你设备上选定的语音引擎；选择在线 Microsoft/Edge 语音并开始朗读时，待朗读文字会发送给 Microsoft。
  • 第三方登录：仅在你主动选择 QQ 或微信登录时调用对应 SDK并处理授权信息。
  • 分享、上传和打开外部应用：仅为完成你主动选择的操作，将内容交给你选择的网站或应用。

请勿向翻译、AI 或其他在线服务提交密码、身份证件、支付信息等敏感内容。你选择境外网站或服务商时，相关信息可能按服务提供地点跨境处理，请先阅读服务商规则。

八、共享、转让、公开披露与跨境

我们不会出售个人信息。仅在完成你主动选择的功能、向本政策列明的接收方传输、履行法定义务、保护用户或公众安全，或取得你另行同意时共享或披露必要信息。发生合并、分立或资产转让并涉及个人信息转移时，我们会依法告知接收方并要求其继续受本政策约束。选择境外网站、AI、翻译、搜索或语音服务时，信息可能跨境传输，接收方及处理规则以你选择的服务为准。

九、保存与删除

本地数据保留到你删除对应记录、使用应用内清理功能、通过系统清除应用数据或卸载应用为止。紫霞服务器访问日志仅在运营、安全和防滥用合理所需期限内保存，之后删除或匿名化。独立网站和第三方服务按其政策保存信息。

紫霞浏览器 App 不创建或保存用户账号，因此没有需要注销的 App 账号。你通过第三方登录访问的网站账号由相应网站管理，应依网站规则申请注销或删除。

十、你的权利及撤回方式

你可以在应用内清除浏览历史、Cookie、缓存、网站权限、下载记录和书架内容；在系统设置中撤销位置、相机、文件和安装应用等权限；在应用设置中关闭搜索建议或停用相关在线功能；也可以清除应用数据或卸载应用以撤回对后续处理的同意。

如需访问、更正、复制或删除由我们控制的个人信息，撤回对本政策的同意，或提出意见和投诉，请发送邮件至下方联系方式。我们会在核验请求后依法处理；撤回不影响撤回前基于同意开展的处理。

十一、未成年人保护

紫霞浏览器是通用浏览器，并非专门面向儿童。不满十四周岁的未成年人应在监护人指导和同意下使用。监护人应关注访问的网站、下载、权限、AI 功能及在线提交的信息；如认为未成年人向我们提供了不适当的信息，请联系我们。

十二、数据安全

我们采取访问控制、传输加密和最小化处理等合理措施减少未经授权的访问、泄露和滥用风险，但互联网传输和本地设备环境无法保证绝对安全。请妥善保管设备、账号和 API 密钥，并从可信来源下载安装包。

十三、政策更新

产品功能、个人信息处理目的、方式、范围或法律要求变化时，我们会更新本政策并公布新版本和日期。发生重大变化时会通过应用内合理方式提示，并在法律要求时重新取得同意。
